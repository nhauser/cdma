Detailed Description

The IArrayUtils interface defines commons need of operations to be perform on a IArray.

Note:
When developing a plug-in consider using the engine's implementation. You should redefine this interface implementation, only in case of very specific needs.

pluginAPI The IArrayUtils aims to provide array's manipulation methods. Essentially shape, memory copy or section reduction operations of the array.
