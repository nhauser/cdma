Detailed Description

Simple implementation that Plugins methods should use for creating compound IDataItem objects.

This implementation is aimed to help plugin developper when special data access methods have to be implemented. Such a method is invoked through the dictionary mechanism when a keyword is not directly mapped whith a path to the data file organization, but is mapped to this method through a <call> tag. Use this implementation only when all data corresponding to the corresponding keyword request may be easely contained in the computer RAM like scalars, spectrum, images value but not stack of hundred or thousands images. For the laters cases consider developping a specific implementation.
