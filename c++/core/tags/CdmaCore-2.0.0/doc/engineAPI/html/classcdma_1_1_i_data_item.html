Detailed Description

Abstraction of a data item container.

It has a DataType, a set of Dimensions that define its Array shape, and optionally a set of Attributes (see IAttribute).

Note:
It must be a base class of data format engine data item class concretization which can be overrided, if needed, by plug-ins based on same engine
