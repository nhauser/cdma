Detailed Description

template<typename CPTR>
class AttributeManager< CPTR >

manages attributes of an object

This type provides an interface to the attribute attached to an object satisfying the IContainer interface. Each type implementing the IContainer interface has a public member variable of type AttributeManager. The type implements basic sequence interface for Python. Thus, one can iterate over such an attribute in python with 

item = ....
for attr in item.attrs:
    print attr

An individual attribute can be picked using the [] operator 

print item.attrs["temperatur"]

If the attribute does not exist a KeyError exception is thrown.
