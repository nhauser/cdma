Detailed Description

DimensionManager class.

Every DataItemWrapper has a public instance of DimensionManager attached to it. This object basically behaves like a list. The length of the list is the number of dimensions of the data item. Each list entry is a list of DimensionWrapper objects which can be associated with the very dimension the list entry belongs to.
